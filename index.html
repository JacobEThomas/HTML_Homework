Summary: Latitude vs Weather Parameters

This project is an exercise that compares various weather parameters with Lattitudes around the world. Coordinates were randomly generated and 666 coorespponding cities were recorded.

Weather parameters were pulled in realtime usine the OpenWeatherAPI.

I used Matplotlib to plot the weather parameters vs. latitude, including temperatue, cloudiness, wind speed, and humidity. This site provides the source data and visualizations created as part of the analysis as well as explanations and descriptions of any trends and correlations witnessed.
